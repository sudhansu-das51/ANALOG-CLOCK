🕰️ Realistic Analog Clock

A simple and visually appealing analog clock built using HTML, CSS, and JavaScript.
This project demonstrates how to create a real-time working clock with smooth hand movements and a clean UI.

🚀 Features
Real-time clock (updates every second)
Hour, minute, and second hands
Minimal and clean design
Center logo support (custom branding)
Pure HTML, CSS, and JavaScript (no libraries)
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
📂 Project Structure
/project-folder
│── index.html
│── style.css
│── script.js
│── /resources (images/logo)
⚙️ How It Works
JavaScript fetches the current time using Date()
Converts time into rotation degrees
Applies rotation using CSS transform
Updates every second using setInterval()
